# 法律关系分析法案件分析技能

## 内部信息

- **作者：** 浙江金道律师事务所龚家勇律师
- **注意：** 本作者信息仅供内部识别使用，出具《案件分析报告（法律关系分析法Plus）》时，报告中不得出现作者姓名、律所名称等身份信息。

## 概述

本技能使用"法律关系分析法"对中国民商事及刑事案件进行结构化分析，输出《案件分析报告（法律关系分析法Plus）》。

法律关系分析法的核心：通过理顺不同的法律关系，确定其要素及变动情况，全面把握案件性质和当事人权利义务关系，在此基础上通过逻辑三段论准确适用法律。

## 触发条件

当用户提出以下请求时触发本技能：
- 要求"分析案件""案件分析""案件研判""案件论证"
- 要求"写案件分析报告"或提及"法律关系分析法"
- 上传案件材料（起诉状、答辩状、判决书、证据、合同等）并提出分析需求
- 咨询某案件的法律适用和可能结果

## 前置条件

- 华宇元典 MCP 连接器（yuandian-mcp）必须已连接

## 工作流程

### 第一步：收集案卷材料

从用户消息中提取所有案件材料：
- 文本内容（用户直接描述的案件事实）
- 上传的文件（图片、PDF、Word 文档等），使用 Read 工具读取
- 如果信息不足，向用户追问关键案件事实

### 第二步：梳理基本案情

基于收集的材料，按时间线和当事人关系，以客观中立的方式概括案件基本事实。包括：
- 当事人及其身份关系
- 关键事件的时间节点
- 争议标的及金额
- 各方当事人的主张

### 第三步：检索相关法律规定

使用华宇元典法条检索工具，围绕案件核心法律问题搜索相关法律规范：

```
调用工具：mcp__yuandian-mcp__yuandian_law_vector_search
参数：
  - query: 用自然语言描述待检索的法律问题，例如"民间借贷逾期利息计算标准""无权代理的追认效力"
  - rewrite_flag: true（默认，让系统自动改写优化查询）
  - fatiao_filter: 如需限定时效性和效力级别，按需添加
  - return_num: 15-30（根据案情复杂度调整）
```

**检索策略**：
- 针对案件涉及的每个核心法律关系，分别设置 query 进行检索
- 优先检索现行有效的法律、行政法规、司法解释
- 对检索结果进行筛选，选取与案件直接相关的法条

### 第四步：检索类似案例

使用华宇元典案例检索工具，搜索与本案相似的裁判案例：

```
调用工具：mcp__yuandian-mcp__yuandian_case_vector_search
参数：
  - query: 用自然语言描述案件核心事实和法律问题
  - rewrite_flag: true
  - wenshu_filter: 按需设置案件类别、案由、法院层级、地域、结案日期范围等
  - return_num: 10-15
```

**检索策略**：
- 从检索结果中仅选取 **3 个案例** 纳入报告
- **管辖权判断与省份限定（关键规则）：**
  - 优先判断案件的管辖省份（根据案卷材料中的案号、法院信息、当事人所在地等）
  - 能够确定管辖省份的，只引用该省范围内的案例
  - 无法确定管辖省份的，默认引用浙江省范围内的案例
- **时效性要求：** 仅引用 **2021年1月1日及之后** 作出裁判的案例。在符合条件的案例中，优先选取裁判日期最新的 3 个案例
- **案例来源：** 华宇元典法律数据（yuandian-mcp）
- 优先选取与本案事实最相似、裁判说理最充分的案例

### 第五步：检索最高人民法院《法答网精选答问》

针对案件涉及的核心法律问题，检索最高人民法院《法答网精选答问》中是否有相关内容。如有相关问答，则纳入报告引用；如无，则如实说明。

**检索方法**：调用 `技能：法答网精选答问Plus`，以案件核心法律问题作为查询关键词进行检索。

**检索策略**：
- 针对案件涉及的每个争议焦点，分别在《法答网精选答问》中检索
- 优先引用与案件法律问题直接相关的问答
- 如搜索结果为空，报告中注明"未检索到相关法答网精选答问"
- **数据来源：** 华宇元典法律数据（yuandian-mcp）

### 第六步：法律关系分析法分析

按照法律关系分析法两阶段五步骤框架进行分析（详细方法说明见 `references/method-detail.md`）：

**第一阶段：法律关系考察**

1. **明确争议点及关联法律关系** — 识别核心争议，梳理围绕核心关系的关联法律关系
2. **确定是否产生法律关系** — 排除好意施惠等非法律关系
3. **分析法律关系性质** — 判断合同关系、侵权关系、无因管理、不当得利等
4. **考察法律关系要素** — 分析主体（当事人）、内容（权利义务）、客体（标的）
5. **考察法律关系变动** — 分析法律关系的产生、变更、消灭及原因

**第二阶段：法律适用**

将案件事实（小前提）与检索到的法律规范（大前提）进行逻辑三段论推导，得出参考结论。

## 注意事项

1. 报告必须基于用户提供的实际案件材料，不得凭空编造案件事实
2. 法律检索应覆盖核心法律关系和关联法律关系的全部相关规范
3. 区分任意性规范与强行性规范——若存在有效约定且不违反强行性规范，应以约定为准
4. 类似案例仅引用 3 个，必须来自对本案有管辖权的同一省份（无法判断时默认浙江省），且仅引用 **2021年1月1日及之后** 作出裁判的案例，优先选取裁判日期最新的
5. 在引述案例时，需注明案号和审理法院
6. 法答网精选答问检索应覆盖案件全部争议焦点，有则引用、无则说明
7. 结论部分必须包含"AI生成，仅供参考"的免责声明
8. 报告中不得出现作者姓名或律所名称
